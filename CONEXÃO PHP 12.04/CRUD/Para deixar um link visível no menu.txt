Para deixar um link visível no menu principal somente para usuários com o login de administrador.

1. Verificação de Sessão no Menu
Supondo que o valor que identifica se o usuário é administrador esteja armazenado no campo tipo na tabela usuarios, você pode verificar isso ao exibir o menu.

2. Exemplo de Código
No arquivo que gera o menu, faça a verificação da sessão e do tipo de usuário. Se o usuário for um administrador, mostre o link para a página exclusiva.


<?php
// Iniciar a sessão
session_start();

// Verificar se o usuário está logado
if (isset($_SESSION['usuario_id'])) {
    // Conectar ao banco de dados
    $conn = new mysqli("localhost", "usuario", "senha", "banco");

    // Verificar conexão
    if ($conn->connect_error) {
        die("Falha na conexão: " . $conn->connect_error);
    }

    // Obter o ID do usuário da sessão
    $usuario_id = $_SESSION['usuario_id'];

    // Consultar o tipo de usuário no banco de dados
    $sql = "SELECT tipo FROM usuarios WHERE id = ?";
    $stmt = $conn->prepare($sql);
    $stmt->bind_param("i", $usuario_id);
    $stmt->execute();
    $res = $stmt->get_result();
    $row = $res->fetch_assoc();

    // Armazenar o tipo do usuário em uma variável
    $usuario_tipo = $row['tipo'];
}
?>

<nav>
    <ul>
        <li><a href="home.php">Home</a></li>
        <li><a href="contato.php">Contato</a></li>

        <?php if (isset($usuario_tipo) && $usuario_tipo == 'admin'): ?>
            <!-- Link visível apenas para administradores -->
            <li><a href="admin.php">Administração</a></li>
        <?php endif; ?>

        <?php if (isset($_SESSION['usuario_id'])): ?>
            <li><a href="logout.php">Sair</a></li>
        <?php else: ?>
            <li><a href="login.php">Login</a></li>
        <?php endif; ?>
    </ul>
</nav>

Iniciar a sessão: A sessão é iniciada para verificar se o usuário está logado.
Consultar o tipo do usuário: Uma consulta ao banco de dados verifica o campo tipo do usuário logado.
Exibição condicional: O link para "Administração" só será exibido se o usuário tiver o tipo 'admin'.
Links de login e logout: Se o usuário estiver logado, o link "Sair" será exibido; caso contrário, o link "Login" aparecerá.
Importante:
Garanta que o tipo do usuário esteja corretamente definido no banco de dados (por exemplo, 'admin' para administradores).
O código assume que você já tem a autenticação configurada e o ID do usuário está armazenado na sessão.
